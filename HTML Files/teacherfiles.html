Uploaded Files From Teachers

    Attention: The files are going to be sent here if they are uploaded by the teacher. If there are no files, then the message "No files uploaded yet" will be shown. They are going to be appear in a list format. The file name will be underlined and the link will download the file if it is clicked. You also have an option to remove the file if you want to. Only click the red cross next to the file extension and you can remove it. The content is about anything concerning students, so please keep an eye on it. It is hugely important.
